Options :
--source <rep> : Répertoire contenant les images à mettre dans la galerie.
--dest <rep> : Répertoire cible (où on va générer le fichier HTML).
--verb : Mode détaillé 
--force : Force la création de vignette, même si la vignette existe déjà.
--help : Affiche la liste des options disponibles
--index <nomFichier> : Permet de spécifier le nom du fichier HTML créé
